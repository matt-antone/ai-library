## The Library

The Library is a private catalog for managing skills, agents, and prompts across devices and teams.

**Full instructions**: Read `SKILL.md` at `LIBRARY_SKILL_DIR` (`~/.claude/skills/library/SKILL.md`) for all commands, cookbook references, source formats, and operational rules.

When using The Library in this environment, execute all operations directly via shell commands. Do not delegate to a `claude` CLI wrapper — read the cookbook file for the operation, then run the git and file operations yourself.
